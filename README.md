# Personal Website — SEG3125 Assignment 1

Portfolio entry page for **SEG3125 — Analysis and Design of User Interfaces** (University of Ottawa).

## Live site
Site is available at:

**https://murtazarhythm.github.io/Personal-Website/**

## Project structure

```
index.html              Main portfolio (About, Experience, Projects, How I Work, Case Studies, Contact)
css/styles.css          Custom theme (colors, typography, layout)
js/main.js              Smooth scroll and active navigation
assets/                 Case study placeholder illustrations (SVG)
case-studies/           “Coming soon” pages for four future designs
```

## Technologies

- HTML5, CSS3, JavaScript
- [Bootstrap 5](https://getbootstrap.com/) (CDN)
- Google Fonts: Fraunces (display), Inter (body)
